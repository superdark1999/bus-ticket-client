## Node & Yarn

Using:
Node: `v16.16.0`
Yarn: `1.22.17`

how to check node version & yarn version
...
node -v
yarn -v
...

## Run app

...
yarn ci
yarn start
...

if something goes wrong:

- Delete yarn.lock
  ...
  yarn
  yarn start
  ...

AdminPage: http://localhost:3000/admin
